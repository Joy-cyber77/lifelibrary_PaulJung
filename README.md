# 정병오 인생 라이브러리

교사 · 기독시민운동가 · 작가 **정병오(Paul Jung)** 의 삶 · 글 · 활동을 담은 디지털 서고.
동생이 오빠를 위해 만든 헌정 사이트.

## 구성
- **홈** — 소개 · 최근 소식(SNS) · 라이브러리 미리보기
- **저서 · 칼럼 · 인터뷰 · 활동 · 어록** — 유형별 아카이브
- **열람실(catalog)** — 전체 자료를 유형 · 연도 · 검색으로 열람

정적 사이트입니다 (HTML / CSS / vanilla JS). 빌드 도구·프레임워크 없음.

## 배포 — Cloudflare Pages
GitHub 저장소를 Cloudflare Pages에 연결하면 push 할 때마다 자동 재배포됩니다.

| 설정 | 값 |
|---|---|
| Framework preset | None |
| Build command | (비움) |
| Build output directory | `/` |

이 저장소의 루트가 곧 사이트입니다.

## 로컬 미리보기
`index.html` 을 브라우저로 엽니다. (폰트는 인터넷 없이 로컬 로드)
